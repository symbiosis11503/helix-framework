👋 歡迎使用 Helix

這是你本地 agent runtime 的主控台。還沒跑過 agent？照下面 3 步驟開始：

  1. CLI 設定 API key：helix login --provider gemini --api-key YOUR_KEY
  2. 編輯 helix.config.js 加 agent 角色定義
  3. 到下方「啟動新 Agent」卡，填 role_id 按 Spawn — 或直接跑 helix repl 開互動對話
