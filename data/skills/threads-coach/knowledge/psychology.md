# 受眾心理學基礎（threads-coach 內建知識庫）

> 補充 algorithm-base.md 的「為什麼這個信號會強」。
> 不是行銷話術，是研究文獻 + 平台行為觀察的歸納。

## 核心原則

演算法是反映「人為什麼會做某件事」的工具，不是創造它的源頭。
要懂演算法，先懂人。

## P1. 為什麼私訊分享比按讚強

按讚是匿名、低成本、無風險的動作。
私訊分享需要：
1. 選一個特定的人
2. 判斷對方會不會喜歡 / 用得到
3. 願意暴露自己的閱讀偏好
4. 承擔對方覺得「無聊」的風險

這是社交資本的小投資。Meta 的權重設計就是反映這個成本差異。

**寫文時**：不要追求「大家都會按讚」的內容，追求「至少有一個人讀完會想 ping 一個朋友」。

## P2. 為什麼 5+ 詞留言比短留言強

寫 5+ 詞的留言代表：
1. 讀者真的讀完了（不是滑過）
2. 觸發了某個想法 / 經驗 / 反駁
3. 願意公開表達

短留言（「+1」「同意」「哈哈」）多半是社交禮貌，不代表內容有觸動。
Meta 的 MSI 計分把這個差異拉到 30:1 不是隨便定的。

**寫文時**：結尾不要設計成「點讚就好」的句子。要設計成「我想補一個 / 反駁一個」的句子。

## P3. 反直覺 vs 共鳴

| | 反直覺 | 共鳴 |
|---|---|---|
| Hook 強度 | 高 | 中 |
| 留言觸發 | 反駁 + 案例 | 「對對對」 |
| 5+ 詞留言比例 | 高 | 低 |
| 私訊分享 | 高（可當談資）| 中 |
| 主信號 | replies + sends | likes |

兩種都有用，但對 B2B / niche 帳號來說，反直覺通常是更值得追求的方向 —
因為它能在小受眾池內製造「值得對話」的密度。

**寫文時**：能寫反直覺就不要寫共鳴。共鳴文是消耗信任，反直覺文是累積信任。

## P4. 為什麼自我表達式留言最有價值

讀者寫「我也遇過 X」「在我這裡是 Y」「如果換成 Z」這類留言時：
1. 他在用你的內容當鏡子，反映自己的經驗
2. 留言區因此變成「集合多個案例的場域」
3. 後來的讀者會看到「這篇文不只是作者觀點，還有 N 個人補充」

這就是 algorithm-base S2 講的「留言區變成第二內容場」。

**寫文時**：結尾的問句要設計成「你怎麼處理 X？」「你看過更極端的嗎？」這類「邀請寫案例」的，不要寫「你覺得呢？」這種沒方向的。

## P5. Niche 帳號的「窄即是深」

對 KOL 帳號，廣度是優勢（觸及多 → 名氣大 → 變現）。
對 niche 帳號，廣度是負擔（觸及陌生人 → 不轉換 → 平台覺得你內容沒匹配上 → 進一步壓 reach）。

**寫文時**：
- 寧願 100 個對的人讀完留言，不要 1000 個路人滑過
- 主題敢窄、立場敢明確、術語敢用
- 受眾自篩是優點，不要為了取悅路人稀釋內容

## P6. 信任複利 > 觸及複利

按讚追求觸及。觸及高 → 演算法給更多 → 更多觸及。但這是脆弱循環：
- 一篇低互動會打斷
- 主題稍偏就崩

留言 / 私訊分享追求信任。信任高 → 老粉穩定回 → 寫深度內容也有人看 → 寫敏感主題也有人挺。
這是穩定累積，不會因一篇崩。

對 niche / B2B 帳號，信任複利是唯一可持續的策略。

## P7. 為什麼「老粉互動」比「路人互動」價值高

Facebook Papers 顯示，平台會追蹤「誰跟你互動」，老粉互動權重明顯比路人高。
這背後的人類行為邏輯：
- 老粉互動代表「持續關注 + 主動回」這是真實的關係訊號
- 路人互動可能是觸發詞、好奇、無聊滑到 — 信號雜訊比高

**寫文時**：不要為了討路人喜歡寫「水化」內容。寫真的會讓老粉想留言的內容。

## P8. 自己回留言為什麼算信號

兩件事：
1. Meta 的留言排序專利顯示，作者回覆會被算成額外的內容信號
2. 你的回覆品質決定留言區會不會繼續延展

如果你只回「謝謝」，留言區就是 dead-end。
如果你接「對，但我發現另一個情境是...」，留言區會被推進。

**寫文後**：把回留言當成第二輪寫文，而不是社交禮貌。

## P9. 為什麼長文不一定差（取決於密度）

US Patent US10404817B2 講的是 Time Spent，不是字數。
500 字的廢話文 < 800 字的密集文。

**寫文時**：
- 每段問自己「這段有沒有比上段多給東西？」
- 沒有就刪
- 不要為了湊長度重複論點換句話說

## P10. 為什麼結尾的開放問句重要

人腦的「未完成感」（Zeigarnik Effect）：
- 一個未答的問題會持續佔用注意力
- 讀者更容易記得 / 回到貼文 / 留言參與

但問句要具體。「你怎麼想？」太模糊，啟動不了未完成感。
「你會把記憶分幾層？」具體到能形成具體答案，啟動感更強。

**寫文時**：結尾問句的判準是「讀者腦中能不能立刻形成具體答案」。
