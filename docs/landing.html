Helix

Local-first AI Agent Framework. Build, run, and govern AI agents on your machine. No cloud required.

npm install -g helix-agent-framework

What You Get

Agent Runtime

Spawn and manage AI agents with persistent session memory, auto-compression, and multi-provider LLM support (Gemini, Claude, OpenAI).

Context OS

Session-grade conversation persistence with FTS search, parent chain checkpoints, and LLM-powered structured summary compression.

Delegation OS

Isolated child agent execution with tool restrictions, recursion depth limits, and batch parallel support.

Execution Safety

35+ dangerous command patterns, Unicode normalization, prompt injection defense, and interceptable hook lifecycle.

Workflow Engine

DAG-based workflow execution with parallel branches, topological sorting, and per-node status tracking.

Tool Registry

Dynamic tool registration with capability binding, role-based access control, and full hook chain integration.

MCP Client

Connect to external MCP servers via stdio transport. Dynamic tool discovery and server lifecycle management.

Memory Manager

Three-tier long-term memory (episodic / semantic / procedural) with importance scoring, time decay, and semantic recall via pgvector (optional).

Trace + Eval

Run/span/metrics tracing with per-run eval score persistence (command-safety / prompt-injection suites), nightly backfill, and badge-ready UI.

Dashboard

Minimal v2 web UI (index + debug) with live health, trace viewer, eval badges, and drop-in debug overlay (press ?).

CLI REPL

helix repl with slash commands: /agents, /memory, /switch, /save, /history, /session, and more.
